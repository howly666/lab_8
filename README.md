# Dota 2 Heroes — TypeScript, React, MUI, Redux, DnD Kit

Практическая работа по TypeScript, React и MUI. Тема: герои Dota 2.

Добавлена страница «Проверь себя»:
- задания на сопоставление;
- задание на сортировку;
- перетаскивание ответов через DnD Kit;
- хранение порядка ответов в Redux Toolkit;
- проверка результата;
- кнопка «Начать снова» со сбросом результатов и перемешиванием ответов.

## Запуск

```bash
npm install
npm run dev
```

Если npm пытается скачать пакеты не с обычного реестра, выполните:

```bash
npm config set registry https://registry.npmjs.org/
npm install
npm run dev
```
